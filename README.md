# keylogger-linux
## Setup

1. Create a virtual environment:
    ```sh
    python3 -m venv myenv
    source myenv/bin/activate
    ```

2. Install dependencies:
    ```sh
    pip install pynput
    ```

3. Run the keylogger:
    ```sh
    python keylogger.py
    ```
